Task: Build a responsive React image slider component from scratch that includes next/previous buttons, indicator dots, and an auto-play feature that safely pauses on hover.

Technical Constraints & Requirements:
No Third-Party Libraries: Do not use Swiper, Slick, Framer Motion, or React-Icons. Use raw React and CSS/Tailwind
Infinite Loop: Clicking "Next" on the last slide must wrap around to the first slide seamlessly (and vice versa for "Previous").
Memory Safety: Ensure the auto-play timer is completely clean and leaves no memory leaks when the component unmounts or state changes.
Semantics: Avoid using unclickable <div> tags for buttons or indicators.
